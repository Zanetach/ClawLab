# ClawLab v0.2.0

Released: 2026-03-24

## Highlights

ClawLab 0.2.0 focuses on making agent operations faster and easier to manage.
This release improves the agent detail experience, simplifies agent creation defaults,
and hardens the production build pipeline for Next.js 16 + Turbopack.

## What's New

### Better agent detail management

- Added editable agent detail views for direct configuration updates
- Moved heavy agent detail loading to the client so the page shell opens faster
- Added lighter agent fetch paths to reduce slow transitions when opening an agent

### Simpler agent creation defaults

- Improved the creation wizard with stronger defaults and generated persona/bootstrap drafts
- Removed unnecessary sub-agent toggles from the UI
- Standardized sub-agent enablement as a default server-side behavior

### Clearer dashboard and list presentation

- Refined dashboard and agent list views for better status and telemetry visibility
- Improved workspace and runtime context shown around agents

### Internal chat foundation

- Added internal chat runtime, streaming APIs, and session plumbing
- Kept the Chat entry disabled in the product UI while the feature remains internal

## Fixes

- Fixed Turbopack NFT tracing warnings in production builds
- Reduced slow page transitions caused by blocking server-side agent detail loading

## Validation

- `npm run lint`
- `npm run build`
